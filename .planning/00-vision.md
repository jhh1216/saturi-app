# 앱 비전 — 사투리 (Saturi)

## 한 줄 요약

대한민국 6개 지역의 방언을 직접 말하며 배우는 음성 인식 기반 사투리 학습 앱

---

## 배경 및 문제 인식

표준어 중심의 교육 환경에서 지역 방언은 점점 사라지고 있다. 젊은 세대는 부모·조부모 세대의 사투리를 이해하지 못하고, 타 지역 방언에 대한 거부감도 높다. 기존 방언 학습 콘텐츠는 텍스트·영상 위주여서 실제 발음을 연습하기 어렵다.

## 해결책

**직접 말하고, 즉시 채점받는** 인터랙티브 방언 학습 경험을 제공한다.

- 사용자가 방언 문장을 보고 따라 말한다
- 음성 인식이 발화를 텍스트로 변환한다
- Levenshtein distance 기반 유사도 알고리즘이 0~100점으로 채점한다
- 피드백 메시지와 점수로 학습 동기를 유지한다

## 대상 사용자

| 사용자 그룹 | 학습 동기 |
|---|---|
| 타 지역 출신 대학생·직장인 | 현지 문화 이해, 소통 개선 |
| 귀향·이민 준비 중인 성인 | 지역 커뮤니티 적응 |
| 방언에 관심 있는 언어 학습자 | 문화·언어 보존 |
| 드라마·영화 팬 | 지역 캐릭터 대사 이해 |

## 지원 지역 (6개)

| 지역 | 이모지 | 대표 특징 |
|---|---|---|
| 서울 | 🏙️ | 표준어 기반, 빠른 억양 |
| 부산 | 🌊 | 경상도 방언, 강한 억양 |
| 전라 | 🌾 | 부드러운 어미, 독특한 말투 |
| 충청 | 🌻 | 느린 리듬, 온화한 말투 |
| 강원 | 🏔️ | 산간 방언, 독특한 어휘 |
| 제주 | 🍊 | 가장 이질적, 고유 어휘 풍부 |

## 핵심 가치

1. **접근성** — 설치 후 즉시 학습 가능, 별도 로그인 없음
2. **즉각적 피드백** — 말하는 순간 채점
3. **문화 보존** — 사라져가는 방언을 디지털로 기록
4. **재미** — 게임처럼 점수를 올리는 학습 경험

## 장기 비전

- 방언별 문장 수 지속 확대
- TTS(flutter_tts)를 활용한 원어민 발음 듣기 기능
- 학습 기록 및 통계 대시보드
- 커뮤니티 기능 (지역민이 직접 예문 제출)
