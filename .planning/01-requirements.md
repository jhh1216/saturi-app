# 요구사항 명세 — MoSCoW 분류

## Must Have (반드시 구현)

### 기능 요구사항

| ID | 요구사항 | 설명 |
|---|---|---|
| M-01 | 지역 선택 화면 | 6개 지역(서울·부산·전라·충청·강원·제주)을 그리드로 표시 |
| M-02 | 방언 문장 목록 | 선택한 지역의 방언 문장 리스트 표시 (방언 + 표준어 번역) |
| M-03 | 음성 녹음 기능 | 마이크 버튼으로 사용자 발화 녹음 (speech_to_text) |
| M-04 | 음성 인식 결과 표시 | 인식된 텍스트를 화면에 실시간 표시 |
| M-05 | 발음 채점 | Levenshtein distance 기반 0~100점 유사도 점수 계산 |
| M-06 | 채점 결과 표시 | 점수 + 피드백 메시지 + 애니메이션으로 결과 표시 |
| M-07 | 재시도 기능 | 녹음 결과 화면에서 다시 도전 가능 |

### 비기능 요구사항

| ID | 요구사항 | 기준 |
|---|---|---|
| M-08 | 플랫폼 지원 | Android, iOS, 웹(Chrome) 빌드 가능 |
| M-09 | 음성 인식 언어 | 한국어(`ko_KR`) 로케일 설정 |
| M-10 | 앱 반응성 | 화면 전환 200ms 이내 |

## 데이터 요구사항

```
방언 문장 (DialectSentence)
├── dialect   : String  — 방언 표현
└── standard  : String  — 표준어 번역

지역 (Region)
├── name      : String  — 지역명
├── emoji     : String  — 대표 이모지
└── sentences : List<DialectSentence>
```

현재 데이터: 6개 지역 × 약 8~10문장 = 약 54개 방언 문장 (lib/data/dialect_data.dart)
