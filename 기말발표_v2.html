ADR-002speech_to_text 패키지
채택
📌 기기 내장 STT 엔진 활용 → 서버/API 키 불필요, 비용 0, 오프라인 동작.
↔ Google STT API 검토 → 유료, 네트워크 의존성 문제
ADR-003Levenshtein distance(레벤슈타인) 알고리즘
채택
📌 방언 어미 변형을 글자 단위로 정확 비교. 순수 Dart 구현, 외부 의존 없음.
↔ 단순 포함 여부 검토 → 정확도 낮음, 부분 점수 불가
